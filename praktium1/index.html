Tentang Website Ini

Website ini dibuat sebagai contoh implementasi HTML dan CSS modern. Menggunakan teknik-teknik CSS terkini seperti Flexbox, Grid Layout, Gradient, Animasi, dan Transisi untuk menciptakan pengalaman visual yang menarik dan responsif di semua perangkat.


Tanpa menggunakan JavaScript, website ini mengandalkan kekuatan CSS untuk menciptakan efek interaktif yang smooth dan modern.
